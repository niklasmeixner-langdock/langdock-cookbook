# A2A Recipes

Recipes for the **Agent-to-Agent (A2A)** protocol — kept separate from `mcp/`
because A2A is a distinct protocol, not an MCP capability.

| Recipe | What it shows |
|---|---|
| [`langdock-a2a-demo/`](langdock-a2a-demo/) | A minimal A2A agent compatible with Langdock's implementation of the A2A protocol. |
